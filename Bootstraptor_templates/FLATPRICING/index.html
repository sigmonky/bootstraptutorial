HOW TO USE

Pricing tables are very easy to use! Boxes class="pricing-table" have five classes are the same as the names of colors: class="dark-blue" , class="blue" , class="green" , class="orange", class="red". To use them just add class name e.g. class="pricing-table green" for green color box.

Standart box has no name of color class just class="pricing-table".

You only need to change the class name of buttons at the bottom of the box. The buttons have a standard Bootstrap classes names btn-primary, btn-warning, btn-danger, btn-info, btn-success.

To resize the box just change the standard Bootstrap class around the box, e.g. class="span2" , class="span3" , class="span4" , etc.

If you need rounded corner just add class class="rounded" to the box, e.g. class="pricing-table rounded"

Just copy and paste the following code to your HTML file 

<!--  STANDART PRICE ITEM  -->
<div class="span3">
<div class="pricing-table"><!-- BODY BOX-->
<div class="pricing-table-header"><!-- HEADER BOX-->
				
<span class="heading">Single applications</span>
<span class="price-value">$<span>14</span><span class="mo">/mo</span></span>
               
</div><!--/ BODY BOX-->
		
<div class="pricing-table-space"></div><!-- CONTENT BOX-->
<div class="pricing-table-features">
<p> <strong>Single</strong> installation</p>
<p><strong> Unlimited </strong> Support</p>
<p><strong>Validation</strong> W3C</p>
<p><strong>Bootstrap</strong> 2.3.0</p>
<p><strong>HTML5</strong> & <strong> CSS3</strong></p>
</div><!-- /  CONTENT BOX-->
		
<div class="pricing-table-sign-up"><!-- BUTTON BOX-->
<a href="#" class="btn btn-large btn-block btn-primary">BUY NOW!</a>
</div><!-- BUTTON BOX-->
		
</div><!--/ BODY BOX-->
</div><!--  SPAN3  -->
<!--  STANDART PRICE ITEM  -->
