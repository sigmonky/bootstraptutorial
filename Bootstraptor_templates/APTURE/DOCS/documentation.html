HTML Structure

Basic HTML scructure for APTURE consists of the building div Bootstrap blocks e.g. class="container" class="row" class="span12" etc. Side menu has a fixed width: 283px, and the content side is indented by the width of 283px Menu value. You can use any Bootstrap elements in template.

Here is the HTML code of index.html

Javascript plugins

Bootsrap Plugins can be included individually (though some have required dependencies), or all at once. Both bootstrap.js and bootstrap.min.js contain all plugins in a single file. Here are the list of all Bootstrap plugins:

  • bootstrap-transition.js
  • bootstrap-alert.js
  • bootstrap-modal.js
  • bootstrap-dropdown.js
  • bootstrap-scrollspy.js
  • bootstrap-tab.js
  • bootstrap-tooltip.js
  • bootstrap-popover.js
  • bootstrap-button.js
  • bootstrap-collapse.js
  • bootstrap-carousel.js
  • bootstrap-typeahead.js
  • bootstrap-affix.js
  • application.js

See more info about all Bootstrap plugins here

Some parts of the plugins for it initiation are in the file custom.js, the other goes in the body of html file.

Some plugins require special code to init them. You will find all of activated pluging parts in file ../assets/js/custom.js . E.g., here is the javascript code to initiated grid preview in index.html 
							
<script>
			jQuery(function() {
				Grid.init();
			});
					  
</script>

Custom small plugin to fixed left menu, when page scroll to bottom 
							
<script>
	var navbox = jQuery('.menu-fix-on-scroll');
		jQuery(window).scroll(function () {
				if (jQuery(this).scrollTop() > 240) {
					navbox.addClass("active");	
				} else {
					navbox.removeClass("active");	
				}
			});
		
</script>
